```bash
pip3 install telepot

```

## Feature

* Base64 Encode / Decode
* URL Encode / Decode
* Search in Baidu
* Search in Wikipedia
* Search in Stack Overflow
* Generate QR Codes (Using Zxing Generator)
* Simple Math Cal Using http://api.mathjs.org
* Timer and Alarm
* ...
